Política de Privacidad de PixelRuler

Última actualización:

1. Información que Recopilamos

PixelRuler es una extensión de navegador que funciona completamente de forma local. No recopilamos, almacenamos ni transmitimos ningún dato personal o información del usuario a servidores externos.

La extensión únicamente:

  • Accede a la actividad del usuario: Detecta clics, movimiento del mouse y scroll únicamente cuando la extensión está activa, para medir elementos web.
  • Accede al contenido de la página web: Lee el DOM y estilos de los elementos únicamente para mostrar información de medición.
  • Almacena preferencias locales: Guarda únicamente en el navegador del usuario las preferencias de idioma y color del overlay mediante chrome.storage.sync.

2. Uso de la Información

Toda la información accedida se procesa exclusivamente en el navegador del usuario. No se envía, vende, comparte ni transfiere ningún dato a terceros o servidores externos.

Las preferencias guardadas (idioma y color) se sincronizan únicamente entre los dispositivos del usuario a través de la cuenta de Chrome, y no son accesibles para nosotros ni para terceros.

3. Permisos de la Extensión

PixelRuler requiere los siguientes permisos:

  • activeTab: Para acceder a la pestaña activa cuando el usuario activa la extensión.
  • scripting: Para inyectar el script necesario para medir elementos.
  • tabs: Para identificar la pestaña activa y comunicarse con el script inyectado.
  • storage: Para guardar las preferencias del usuario (idioma y color).

Todos estos permisos se utilizan exclusivamente para el funcionamiento de la extensión y no para recopilar datos personales.

4. Datos de Terceros

No vendemos, transferimos ni compartimos datos con terceros. PixelRuler no utiliza servicios de terceros para analítica, publicidad o cualquier otro propósito.

5. Seguridad

Dado que PixelRuler funciona completamente de forma local y no transmite datos, no hay riesgo de interceptación o acceso no autorizado a información del usuario.

6. Cambios en esta Política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será reflejado en esta página con la fecha de última actualización.

7. Contacto

Si tienes preguntas sobre esta política de privacidad, puedes contactarnos a través del repositorio del proyecto en GitHub o la página de la extensión en Chrome Web Store.
